## Amazon Prime Mastercardの基本スペック

三井住友カードが発行するAmazon提携カード。プライム会員であれば「Amazon Prime Mastercard」、非会員であれば「Amazon Mastercard」として発行される。カードの基本スペックは共通だが、Amazonでの還元率がプライム会員かどうかで異なる。

| 項目 | 内容 |
|---|---|
| 年会費 | 永年無料 |
| 申込対象 | 満18歳以上（高校生除く） |
| 基本還元率 | 1.0%（200円につき1ポイント） |
| ポイント名 | Amazonポイント |
| 国際ブランド | Mastercard |
| 家族カード | 無料（3枚まで） |
| ETCカード | 初年度無料（年1回利用で翌年も無料、利用なしの場合550円） |

ポイントはAmazonポイントとして自動的に貯まり、Amazon.co.jpでの買い物に1ポイント=1円で使える。ポイント交換の手間がかからない点がこのカードの利点。

## ポイント還元率の仕組み

| 利用先 | 還元率 | 条件 |
|---|---|---|
| Amazon.co.jp（プライム会員） | 2.0% | プライム会員であること |
| Amazon.co.jp（非プライム会員） | 1.5% | — |
| コンビニ（セブン・ローソン・ファミマ等） | 最大7% | スマホのMastercardタッチ決済 |
| その他の利用 | 1.0% | — |

コンビニでの最大7%還元は、三井住友カード共通の特典。スマホのMastercardタッチ決済（Apple Pay / Google Pay）が条件で、物理カードのタッチ決済では最大5%になる。

## Amazonでの還元率2.0%

プライム会員であれば、Amazon.co.jpでの購入に対して2.0%のAmazonポイントが還元される。年間のAmazon利用額が20万円の場合、4,000ポイント（4,000円相当）が貯まる計算。

プライム会員でなくても1.5%還元のため、Amazonでの買い物がメインであれば基本還元率1.0%のカードより効率が良い。

ただし、Amazon以外の利用では1.0%還元。楽天カード（楽天市場3.0%〜）やJCB CARD W（Amazon 2.0%）と比較すると、Amazon専用カードとしての位置づけが明確。

## プライム会員との関係

| 項目 | Amazon Prime Mastercard | Amazon Mastercard |
|---|---|---|
| プライム会員 | 必要 | 不要 |
| Amazon還元率 | 2.0% | 1.5% |
| その他の還元率 | 1.0% | 1.0% |
| 年会費 | 無料 | 無料 |

カード自体の年会費は両方とも無料。プライム会費（年額5,900円 or 月額600円）は別途必要。プライム会員を解約すると、カードは自動的にAmazon Mastercardに切り替わり、Amazon還元率が1.5%に下がる。

## コンビニ最大7%還元の仕組み

三井住友カード発行のため、対象のコンビニ・飲食店でのスマホタッチ決済で最大7%還元が適用される。これは三井住友カード ゴールド(NL)と同じ特典。

| 決済方法 | 対象コンビニ・飲食店での還元率 |
|---|---|
| スマホタッチ決済（Apple Pay / Google Pay） | 最大7% |
| 物理カードのタッチ決済 | 最大5% |
| ICチップ差し込み | 通常還元率 |

セブン-イレブン、ローソン、ファミリーマート、マクドナルド等が対象。コンビニ利用が多い場合は年会費無料カードとしては高い還元率を得られる。

## 注意点

| 項目 | 内容 |
|---|---|
| ブランド | Mastercardのみ（Visa非対応） |
| ポイント | Amazonポイント限定（他社ポイントや現金への交換不可） |
| ETCカード | 年1回利用しないと翌年550円 |
| 旅行保険 | なし |
| ショッピング保険 | 年間200万円まで（海外利用・リボ・分割3回以上） |

ポイントがAmazonポイント限定のため、Amazonを使わなくなった場合にポイントの使い道がなくなる。また、旅行傷害保険が付帯しない点は、他のゴールドカードやプラチナカードとの大きな違い。

## JCB CARD Wとの比較

Amazon還元率2.0%のカードとしてJCB CARD Wが競合する。

| 項目 | Amazon Prime Mastercard | JCB CARD W |
|---|---|---|
| 年会費 | 無料 | 無料 |
| Amazon還元率 | 2.0%（プライム会員） | 2.0%（ポイントアップ登録） |
| 基本還元率 | 1.0% | 1.0% |
| ブランド | Mastercard | JCB |
| ポイント | Amazonポイント | Oki Dokiポイント |
| 申込年齢 | 18歳以上 | 18〜39歳 |

Amazon還元率は同率2.0%。JCB CARD Wはプライム会員でなくても2.0%が適用される代わりに、39歳以下限定の年齢制限がある。Amazon Prime Mastercardは年齢制限がなく、Amazonポイントが直接貯まる手軽さがある。

## まとめ

Amazon Prime Mastercardは、Amazonをメインの買い物先にしているプライム会員にとって最適なカード。年会費無料・Amazon 2.0%・コンビニ最大7%の組み合わせは、日常利用のカードとして十分な性能。ポイントがAmazon限定である点を許容できるかが判断基準になる。
