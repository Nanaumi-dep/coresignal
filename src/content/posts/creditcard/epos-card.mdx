## エポスカードの基本スペック

エポスカードはマルイグループが発行する年会費永年無料のクレジットカード。国際ブランドはVisa。基本還元率0.5%（200円＝1エポスポイント）は一般的な水準だが、年会費無料カードとしては海外旅行保険の補償額が突出している。

| 項目 | 内容 |
|---|---|
| 年会費 | 永年無料 |
| 国際ブランド | Visa |
| 基本還元率 | 0.5%（200円＝1ポイント） |
| ポイント有効期限 | 2年間 |
| ETCカード | 無料 |
| 家族カード | なし |
| 発行スピード | エポススマホカード最短5分 / マルイ店頭で即日発行 |

## 海外旅行保険が年会費無料カードで最高水準

エポスカードの最大の特徴は、年会費無料ながら海外旅行傷害保険が付帯していること。2023年10月以降、利用付帯に変更されたが、補償額は据え置きで維持されている。

| 補償項目 | 補償額 |
|---|---|
| 傷害死亡・後遺障害 | 最高3,000万円 |
| 傷害治療費用 | 1事故あたり最高200万円 |
| 疾病治療費用 | 1疾病あたり最高270万円 |
| 賠償責任 | 1事故あたり最高3,000万円（免責なし） |
| 救援者費用 | 最高100万円 |
| 携行品損害 | 最高20万円（免責3,000円） |

疾病治療費用270万円は、年会費無料のクレジットカードでは最高水準。海外での急な体調不良で最も使う可能性が高いのが傷害治療・疾病治療の補償であり、この金額が充実しているのは実用面で大きい。

利用付帯の条件は、旅行代金（ツアー料金・航空券・空港までの公共交通機関）をエポスカードで支払うこと。旅行前に航空券や空港バスの料金をカードで決済すれば条件を満たせる。

## マルコとマルオの7日間で10%OFF

年4回開催される「マルコとマルオの7日間」では、マルイ・モディでの買い物がエポスカード決済で10%OFF。マルイのセール対象外商品も割引対象になる。マルイ通販「マルイウェブチャネル」での買い物も10%OFF。

マルイで年間5万円の買い物をする場合、10%OFFで5,000円分の割引。これだけで基本還元率0.5%の約20倍に相当する実質還元を受けられる計算になる。

## たまるマーケットでネット通販のポイントが2〜30倍

エポスポイントUPサイト「たまるマーケット」を経由してネットショッピングすると、ポイントが最大30倍（15%還元相当）になる。対象ショップは450店舗以上。

たまるマーケットのポイント倍率は、通常ポイント（200円＝1ポイント）を含んだ表記。2倍なら実質1.0%還元、6倍なら実質3.0%還元になる。

## 全国1万店以上の優待

エポトクプラザでは、全国1万店以上の飲食店・レジャー施設・カラオケ・映画館などで割引や優待を受けられる。

代表的な優待として、イオンシネマの映画鑑賞券が1,400円（通常1,800円）、カラオケ館の室料30%OFF、富士急ハイランドのフリーパス割引など。日常的に使える優待が多いため、年会費無料カードとしてはコストパフォーマンスが高い。

## ゴールドインビテーションが最大の魅力

エポスカードを年間約50万円利用すると、エポスゴールドカードへのインビテーション（招待）が届く。インビテーション経由で切り替えると、ゴールドカードの年会費が永年無料になる。

通常申込みの場合、エポスゴールドカードの年会費は5,000円（税込）。年間50万円以上の利用で翌年以降は永年無料になるが、初年度は必ず年会費がかかる。インビテーション経由なら初年度から無料。

エポスゴールドカードの詳しいスペックは[エポスゴールドカードの還元率・特典を整理する](/creditcard/epos-gold/)で整理している。

ゴールドカードで年間100万円以上を利用すると、さらにエポスプラチナカードへのインビテーションも届く。通常カードからゴールド、プラチナへとステップアップできる導線が設計されている点は、長期運用を前提としたクレジットカード選びで大きな魅力になる。

## EPOS CRYPTOカード for bitbank

## まとめ

エポスカードの実力は、年会費無料という前提で見ると際立っている。海外旅行保険の疾病治療270万円、マルコとマルオの10%OFF、全国1万店以上の優待。そしてゴールドインビテーションによる年会費永年無料のゴールドカードへのステップアップ。

基本還元率0.5%は他の年会費無料カードと比べて突出しているわけではない。だが、「まず無料で持って、使い続けることでゴールドの永年無料権を獲得する」という設計は、長期運用を前提とした合理的な選択肢になる。
