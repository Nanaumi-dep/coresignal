## EPOS CRYPTOカード for bitbankとは

2026年4月27日、エポスカードと暗号資産取引所bitbankの業務提携により「EPOS CRYPTOカード for bitbank」が発行開始された。暗号資産取引所の残高からクレジットカード利用代金を引き落とせる日本初のクレジットカードとなる。

通常のエポスカードとの最大の違いは、ポイント還元がエポスポイントではなく暗号資産で行われる点と、引落し先にbitbank口座を選択できる点の2つ。

## 基本スペック

| 項目 | 内容 |
|---|---|
| カード名 | EPOS CRYPTOカード for bitbank |
| 発行会社 | 株式会社エポスカード |
| 国際ブランド | Visa |
| 年会費 | 永年無料 |
| 還元率 | 0.5%（暗号資産での還元） |
| 還元対象銘柄 | BTC（ビットコイン）/ ETH（イーサリアム）/ ASTR（アスター）から選択 |
| 引落し方法 | 金融機関口座 or bitbank口座（毎月選択可） |
| 引落し対応暗号資産 | ビットコイン（BTC）のみ |
| カードデザイン | ナンバーレス（1種類） |
| 申込条件 | bitbank口座開設済み＋本人確認完了 |
| 発行開始日 | 2026年4月27日 |

## 暗号資産での還元の仕組み

通常のエポスカードが200円につき1エポスポイント（還元率0.5%）を付与するのに対し、EPOS CRYPTOカードは同じ還元率0.5%で暗号資産が付与される。

還元される暗号資産は以下の3銘柄から選択できる。

| 銘柄 | ティッカー | 特徴 |
|---|---|---|
| ビットコイン | BTC | 時価総額1位。長期保有の定番 |
| イーサリアム | ETH | 時価総額2位。スマートコントラクト基盤 |
| アスター | ASTR | 日本発のWeb3プロジェクト。国内での注目度が高い |

複数銘柄から選択できるクレジットカードの暗号資産還元は日本初とされている。

## 暗号資産での引落し

もう一つの大きな特徴が、カード利用代金の引落し先としてbitbank口座を選択できること。引落し方法は毎月bitbankのマイページから「金融機関口座」か「bitbank口座」かを切り替えられる。

ただし、2026年5月時点で引落しに対応している暗号資産はビットコイン（BTC）のみ。引落し日のレートで日本円換算され、相当額のBTCがbitbank口座から差し引かれる。

ビットコインの価格変動リスクがある点は理解しておく必要がある。上昇局面ではBTCの減少量が少なくて済むが、下落局面では多くのBTCが消費される。

## SBI VISAクリプトカードとの比較

暗号資産連携のクレジットカードとしては、住信SBIネット銀行の[SBI VISAクリプトカード](/creditcard/sbi-visa-crypto-card/)も存在する。

| 項目 | EPOS CRYPTO for bitbank | SBI VISAクリプトカード |
|---|---|---|
| 年会費 | 無料 | 無料 |
| 還元率 | 0.5%（暗号資産） | 0.5%（XRP） |
| 選択可能銘柄 | BTC / ETH / ASTR | XRP固定 |
| 暗号資産引落し | BTC引落し可 | なし |
| 連携取引所 | bitbank | SBI VCトレード |
| 国際ブランド | Visa | Visa |

最大の違いは還元銘柄の選択肢と引落し機能の有無。EPOS CRYPTOカードはBTC・ETH・ASTRの3銘柄から選べて、さらにBTC引落しにも対応している。一方、SBI VISAクリプトカードはXRP固定だが、SBI経済圏との連携が強み。

どちらが合うかは、利用している取引所と欲しい銘柄で決まる。bitbankユーザーでBTCかETHを貯めたいならEPOS CRYPTO、SBI VCトレードユーザーでXRPを貯めたいならSBI VISAクリプトカードという切り分けになる。

## 申込みの流れ

1. bitbankで口座開設（未開設の場合）
2. bitbankで本人確認（KYC）を完了
3. エポスカード公式サイトからEPOS CRYPTOカードを申込み
4. bitbankアカウントとカードを連携

bitbank口座の開設と本人確認が完了していることが申込みの前提条件になる。

## まとめ

EPOS CRYPTOカード for bitbankは、年会費無料で暗号資産還元と暗号資産引落しを両立した日本初のクレジットカード。BTC・ETH・ASTRの3銘柄から還元先を選べる点と、毎月の引落しをBTCで行える点が特徴的。

暗号資産に興味があるが積極的にトレードするほどではない層にとって、日常の買い物でBTCやETHが自動的に貯まる仕組みは手軽な入口になる。通常のエポスカードとの使い分けや、[エポスゴールドカード](/creditcard/epos-gold/)へのステップアップも含めて、自分のカード戦略に合うかどうかで判断すればよい。
