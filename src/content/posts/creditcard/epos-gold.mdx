## エポスゴールドカードの基本スペック

マルイグループのエポスカードが発行するVisaゴールドカード。通常年会費5,000円（税込）だが、年間50万円の利用で翌年以降の年会費が永年無料になる。エポスカードからのインビテーション（招待）経由で申し込めば、初年度から年会費が永年無料になるルートもある。

| 項目 | 内容 |
|---|---|
| 年会費 | 5,000円（税込）※年間50万円利用 or インビテーションで永年無料 |
| 申込対象 | 20歳以上（学生除く）で安定した収入がある方 |
| 基本還元率 | 0.5%（200円につき1ポイント） |
| ポイント名 | エポスポイント |
| 国際ブランド | Visa |
| 家族カード | エポスファミリーゴールド（紹介制・永年無料） |
| ETCカード | 無料 |
| Apple Pay | 対応 |

国際ブランドはVisaのみ。Mastercard・JCBは選択できないが、Visaは国内外の加盟店数で圧倒的なカバー率を持つため、メインカードとして不便を感じる場面は少ない。

## ポイント還元率の仕組み

基本還元率は0.5%だが、3つの仕組みを組み合わせることで実質還元率を引き上げられる設計になっている。

| 利用先 | 還元率 | 条件 |
|---|---|---|
| 通常利用 | 0.5% | 200円につき1pt |
| マルイ・モディ | 1.0% | 通常の2倍 |
| 選べるポイントアップショップ | 最大1.0% | 3店舗まで登録可（通常の最大2倍） |
| 年間50万円ボーナス | +0.5%相当 | 2,500pt付与（50万円に対して） |
| 年間100万円ボーナス | +1.0%相当 | 10,000pt付与（100万円に対して） |

基本還元率0.5%だけを見ると高還元とは言いがたいが、年間ボーナスポイントと選べるポイントアップショップを組み合わせた総合力がこのカードの本領。

## 選べるポイントアップショップ

エポスゴールドカード最大の特徴といえるのがこの制度。300以上のショップから最大3つを登録でき、登録したショップでの還元率が最大2倍（1.0%）になる。

対象には日常利用の店舗だけでなく、公共料金や保険料も含まれる。

| ジャンル | 主な対象ショップ |
|---|---|
| スーパー | イトーヨーカドー、イオン、西友、ライフ |
| コンビニ | セブン-イレブン |
| ドラッグストア | マツモトキヨシ、ウエルシア |
| 公共料金 | 東京電力、東京ガス、NHK |
| 通信 | NTTドコモ、au、ソフトバンク |
| 保険 | 国民年金 |
| 交通 | JR東日本、モバイルSuica |

登録は3か月ごとに変更可能。固定費の支払いを集中させれば、基本還元率の低さを十分にカバーできる。国民年金をポイントアップショップに登録して支払えば、年間約20万円の支払いが1.0%還元になる計算。

## 年間ボーナスポイント

年間利用額に応じてボーナスポイントが付与される。通常ポイントとの合算で見ると、100万円ちょうどの利用が最も効率的。

| 年間利用額 | 通常ポイント | ボーナス | 実質還元率 |
|---|---|---|---|
| 50万円 | 2,500pt | 2,500pt | 1.0% |
| 100万円 | 5,000pt | 10,000pt | 1.5% |
| 150万円 | 7,500pt | 10,000pt | 1.17% |

100万円を超えた分は0.5%還元のみとなるため、100万円を超える利用がある場合はサブカードとの使い分けが有効。三井住友カード ゴールド(NL)も同様の「100万円で最大効率」構造だが、エポスゴールドカードは50万円で年会費無料になるハードルの低さが差別化ポイント。

## 年会費を無料にする2つのルート

エポスゴールドカードの年会費を永年無料にするルートは2つある。

| ルート | 条件 | 年会費 |
|---|---|---|
| インビテーション経由 | エポスカード利用実績で招待 | 初年度から永年無料 |
| 直接申込 | 年間50万円以上利用 | 翌年以降永年無料 |

インビテーションの条件は公式に明記されていないが、エポスカード（一般）で年間50万円程度の利用があると届くとされている。半年〜1年程度で届いたケースが多く報告されている。審査はインビテーション経由であればほぼ通過するため、まず一般のエポスカード（年会費無料）を作り、インビテーションを待つのがコスト面では最も合理的。

## エポスファミリーゴールド

エポスゴールドカード・プラチナカード会員が二親等以内の家族を紹介することで、家族も年会費永年無料でゴールドカードを持てる制度。一般的な「家族カード」とは異なり、家族それぞれに独立したゴールドカードが発行される。

| 項目 | 内容 |
|---|---|
| 年会費 | 永年無料（紹介制） |
| 対象 | 二親等以内の家族（18歳以上） |
| カード仕様 | エポスゴールドカードと同等 |
| ポイント共有 | 家族間でポイントをシェア可能 |

家族それぞれが選べるポイントアップショップを3店舗登録でき、ボーナスポイントも個別に付与される。家族合算での年間利用額に応じたファミリーボーナスも別途ある。

## 海外旅行傷害保険

エポスゴールドカードの保険はクレジットカード付帯としては手厚い部類に入る。

| 補償項目 | 補償額 |
|---|---|
| 傷害死亡・後遺障害 | 最高5,000万円 |
| 傷害治療費用 | 最高300万円 |
| 疾病治療費用 | 最高300万円 |
| 賠償責任 | 最高5,000万円 |
| 携行品損害 | 最高50万円（免責3,000円） |
| 救援者費用 | 最高100万円 |

利用付帯（旅行代金をカードで支払った場合に適用）。治療費用が傷害・疾病ともに300万円と、年会費無料クラスのゴールドカードとしては高水準。[三井住友カード ゴールド(NL)](/creditcard/smbc-gold-nl/)の海外旅行保険は最高2,000万円のため、旅行保険の充実度ではエポスゴールドカードに分がある。

## 空港ラウンジ

国内主要空港およびハワイ・ダニエル・K・イノウエ国際空港のラウンジを無料で利用できる。対象は国内16空港＋ハワイ。ゴールドカード付帯の空港ラウンジサービスとしては標準的な内容。

## 他のゴールドカードとの比較

| 項目 | エポスゴールド | 三井住友ゴールド(NL) | JCBゴールド |
|---|---|---|---|
| 年会費 | 5,000円（50万円で無料） | 5,500円（100万円で無料） | 11,000円 |
| 基本還元率 | 0.5% | 0.5% | 0.5% |
| 年間100万円時 | 1.5% | 1.5% | 0.5%（＋JCBスターメンバーズ） |
| 国際ブランド | Visa | Visa / Mastercard | JCB |
| 海外旅行保険 | 最高5,000万円 | 最高2,000万円 | 最高1億円 |
| 空港ラウンジ | ○ | ○ | ○ |
| 年会費無料のハードル | 50万円 | 100万円 | なし |

年会費無料のハードルが50万円と低いのがエポスゴールドカードの優位点。三井住友カード ゴールド(NL)は100万円必要なため、利用額が年50〜100万円の層にはエポスゴールドカードの方が相性が良い。一方、コンビニ・飲食店での還元率は三井住友カード ゴールド(NL)が最大7%と圧倒的に高いため、日常の利用先によって最適解は変わる。

## エポスポイントの使い道

エポスポイントは1ポイント＝1円として利用できる。主な使い道を整理する。

| 使い道 | レート |
|---|---|
| マルイでの買い物 | 1pt = 1円 |
| プリペイドカードチャージ | 1pt = 1円 |
| Amazonギフト券 | 1pt = 1円 |
| dポイント・Pontaポイント等 | 1pt = 1pt（一部0.5pt） |
| JAL/ANAマイル | 1pt = 0.5マイル |

等価交換できる使い道が多く、ポイントの使い勝手は良い部類。プリペイドカード（エポスVisaプリペイドカード）にチャージすればVisa加盟店で使えるため、実質的に現金と同等の価値になる。

## まとめ

エポスゴールドカードは「年間50万円で年会費無料」「選べるポイントアップショップで固定費の還元率アップ」「年間100万円で実質1.5%還元」という3つの仕組みが噛み合った設計になっている。旅行保険の手厚さも含めると、年会費無料のゴールドカードとしてはトップクラスのコストパフォーマンスである。

まずエポスカード（一般・年会費無料）を発行し、インビテーションを経て年会費永年無料のゴールドカードを手に入れるのが最も合理的なルート。マルイでの買い物が多い人はもちろん、選べるポイントアップショップに対象ショップがある人にとっては、メインカードの有力候補になる。
