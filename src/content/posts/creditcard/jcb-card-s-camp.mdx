## キャンペーン概要

JCBカードSの新規入会キャンペーンが2026年3月31日で終了する。対象優待店での20%還元、スマホ決済20%還元、利用額キャッシュバック、家族カード特典の4つを組み合わせると、最大25,000円相当の特典が得られる。

年会費無料のカードで25,000円相当は、入会キャンペーンとしてかなり大きい。ただし各キャンペーンには個別の条件と注意点があるため、一つずつ整理する。

## キャンペーン詳細

### ① 対象優待店で最大20%還元（最大8,000pt）

| 項目 | 内容 |
|---|---|
| 対象 | Amazon.co.jp、スターバックス、マクドナルド、ガストなど80以上の優待店 |
| 仕組み | 通常のポイントアップ登録（最大10%）＋新規入会限定上乗せ（10%） |
| 上限 | 新規入会上乗せ分：最大8,000ポイント |
| 条件 | MyJCBアプリで対象優待店をポイントアップ登録してから利用 |
| キャンペーンページ | [JCB公式：対象優待店の利用で最大20%還元](https://www.jcb.co.jp/campaign/jpartner2601/index.html) |

ポイントアップ登録を忘れると通常還元率のまま。カードが届いたら、利用前にMyJCBアプリで使いたい店舗を登録しておくこと。

### ② スマホ決済20%ポイント還元（最大12,000pt）

| 項目 | 内容 |
|---|---|
| 対象 | Apple Pay・Google Pay・Samsung Wallet・MyJCB Pay |
| 還元率 | スマホ決済利用金額の20% |
| 上限 | JCBカードSは最大12,000ポイント（＝60,000円利用で上限到達） |
| 利用対象期間 | 3月入会の場合：2026年3月1日〜5月15日 |
| ポイント付与 | 3月入会の場合：6月下旬頃 |
| キャンペーンページ | [JCB公式：スマホ決済で20%ポイント還元](https://www.jcb.co.jp/campaign/20pb2601/index.html) |

カード現物でのタッチ決済は対象外。Apple PayやGoogle Pay経由でのタッチ決済のみが対象。オンラインショッピングも対象外。この違いを把握していないと「タッチ決済したのにポイントがつかない」となりかねない。

### ③ 利用額キャッシュバック（1,000円）

| 項目 | 内容 |
|---|---|
| 条件 | 入会月から翌々月15日までに合計3万円以上利用 |
| 特典 | 1,000円キャッシュバック |

### ④ 家族カード特典（最大4,000円）

| 項目 | 内容 |
|---|---|
| 条件 | 家族カードを追加申込 → キャンペーン参加登録 → 1回以上利用 |
| 特典 | 家族1名につき2,000円キャッシュバック（最大2名＝4,000円） |

## 4つの合計

| キャンペーン | 最大金額 |
|---|---|
| ① 対象優待店20%還元 | 8,000pt |
| ② スマホ決済20%還元 | 12,000pt |
| ③ 利用額キャッシュバック | 1,000円 |
| ④ 家族カード | 4,000円 |
| **合計** | **最大25,000円相当** |

①②を満額達成するには対象優待店で80,000円＋スマホ決済で60,000円の利用が必要。③の1,000円はほぼ確実に取れるが、④は家族カードを発行できる人に限られる。

## 対象外の人

2026年1月12日時点で対象カード（JCBカードS、W、W plus L、ゴールド、プラチナ）をすでに保有している場合、このキャンペーンの対象外。2025年1月以降に一度でも対象カードに入会した人も対象外となる。

JCBカードSからJCBゴールドにアップグレードした場合などは注意が必要。

## JCBカードSの基本スペック

| 項目 | 内容 |
|---|---|
| 年会費 | 永年無料 |
| 還元率 | 0.5%（1,000円につき1ポイント＝5円相当） |
| 国際ブランド | JCB |
| 付帯保険 | 海外旅行傷害保険（最高2,000万円・利用付帯） |
| スマホ決済 | Apple Pay・Google Pay対応 |
| ナンバーレス | 対応（カード番号がカード面に印字なし） |
| 特典 | JCB CARD Wと異なり年齢制限なし |

## 注意点

MyJCBアプリへのログインが複数のキャンペーンで条件に含まれている。カードが届いたら、利用開始前にまずアプリをインストールしてログイン、そして対象優待店のポイントアップ登録を済ませておくこと。ブラウザ版MyJCBではなくアプリ版でないと対象にならない。

また、①の対象優待店還元と②のスマホ決済還元は別キャンペーン。Amazonでの買い物をApple Pay経由で行った場合、両方に該当するかどうかはJCB公式で確認が必要。

## まとめ

JCBカードSの入会キャンペーンは3月31日で終了。年会費無料で最大25,000円相当の特典が得られる。JCBカードSと[JCB CARD W](/creditcard/jcb-card-w/)のどちらを選ぶべきかは[比較記事](/creditcard/jcb-card-s-vs-card-w/)で整理している。特にスマホ決済を日常的に使う人、Amazon・スタバなど対象優待店の利用が多い人にとってメリットが大きい。

ただし「ポイントアップ登録」「MyJCBアプリログイン」「スマホ決済限定（カード現物は対象外）」など、条件を正確に把握していないと想定した還元が受けられない。申し込む前に各キャンペーンページで条件を確認しておくこと。
