## JCB CARD Wの基本スペック

JCBが発行する39歳以下限定の高還元カード。年会費無料で基本還元率1.0%と、JCBのプロパーカードとしては最も還元率が高い。

| 項目 | 内容 |
|---|---|
| 年会費 | 永年無料 |
| 申込対象 | 18歳〜39歳（40歳以降も継続利用可） |
| 基本還元率 | 1.0%（1,000円につき2 Oki Dokiポイント） |
| ポイント名 | Oki Dokiポイント |
| 国際ブランド | JCB |
| 家族カード | 無料 |
| ETCカード | 無料 |

39歳以下で申し込めば、40歳以降もカードを保有し続けられる。年齢制限は申込時のみで、一度発行されれば更新時に年齢で弾かれることはない。

## ポイント還元率の仕組み

JCB CARD Wの還元率は「通常のJCBカードの2倍」が基本設計。さらにJCB ORIGINAL SERIESパートナーでの利用で上乗せされる。

| 利用先 | 還元率 | 条件 |
|---|---|---|
| 通常利用 | 1.0% | — |
| Amazon.co.jp | 2.0% | ポイントアップ登録（無料） |
| セブン-イレブン | 2.0% | ポイントアップ登録（無料） |
| スターバックス | 最大10.5% | ポイントアップ登録（無料） |
| メルカリ | 2.0% | ポイントアップ登録（無料） |
| Oki Dokiランド経由 | 1.0%〜 | 経由してネットショッピング |

ポイントアップ登録はJCBの会員サイト（MyJCB）から無料で行える。登録しないと通常還元率のままになるため、カード発行後に忘れず設定する必要がある。

## Amazonでの還元率2.0%

JCB CARD WでAmazonを利用すると還元率2.0%。年間のAmazon利用額が20万円の場合、4,000円分のポイントが貯まる計算。

Amazon Prime Mastercard（プライム会員2.0%）と同率だが、JCB CARD Wはプライム会員でなくても2.0%が適用される点が異なる。プライム年会費（5,900円）を払っていない場合は、JCB CARD Wの方が実質的にお得になる。

## 注意点

| 項目 | 内容 |
|---|---|
| 申込年齢 | 39歳以下限定。40歳以上からは申込不可 |
| ポイント付与単位 | 1,000円単位（999円以下は切り捨て） |
| 国際ブランド | JCBのみ（Visa / Mastercard非対応） |
| ショッピング保険 | 海外のみ対応（国内は対象外） |

ポイント付与が1,000円単位のため、少額決済が多い場合はポイントが付かない端数が積み重なる。公共料金やサブスクリプションなど、まとまった金額の決済に使う方が効率的。

また、国際ブランドがJCBのみのため、海外での利用可能店舗はVisa / Mastercardと比べて少ない。国内利用がメインであれば問題ないが、海外利用が多い場合はサブカードの併用が前提になる。

## まとめ

JCB CARD Wは、39歳以下であれば最初に持つべきカードの候補に入る。年会費無料・基本還元率1.0%・Amazon 2.0%という組み合わせは、年会費無料カードの中では最高水準。40歳以上の場合や優待・保険を重視する場合は、[JCBカードSとの比較記事](/creditcard/jcb-card-s-vs-card-w/)も参考にしてほしい。JCBブランドの制約を受け入れられるなら、メインカードとして十分機能する。
