## JCB THE CLASSの基本スペック

JCBオリジナルシリーズの最高峰に位置するブラックカード。申込制ではなく、JCBからのインビテーション（招待）を受けた人だけが発行できる。

| 項目 | 内容 |
|---|---|
| 年会費 | 55,000円（税込） |
| 申込方法 | インビテーションのみ（直接申込不可） |
| 家族カード | 無料（8枚まで） |
| ETCカード | 無料 |
| 基本還元率 | 0.5%（Oki Dokiポイント） |
| 国際ブランド | JCB |
| 旅行傷害保険 | 海外最高1億円・国内最高5,000万円 |

年会費55,000円はプラチナカード（27,500円）の2倍だが、メンバーズ・セレクションだけで25,000円相当の価値がある。実質的な年会費負担は特典の使い方次第で大きく変わる。

## 主要特典

### メンバーズ・セレクション

年に1度、JCBが厳選した商品やサービスの中から1つを選べる特典。ザ・クラス会員限定。

| コース | 内容 | 価値の目安 |
|---|---|---|
| 厳選ギフトコース | 食料品・家具・家電など50〜60品から選択 | 約25,000円相当 |
| エンターテインメント＆トラベルコース | JTB旅行券（国内2万円分 or 海外3万円分）、テーマパークチケット（ペア）等 | 約20,000〜30,000円相当 |
| ロイヤルコース（2026年〜） | 年間300万円以上利用者限定の上位コース | — |

テーマパークのペアチケットは、通常では入手困難な特典として人気が高い。

### グルメ・ベネフィット

国内の対象レストランで所定のコース料理を2名以上で予約すると、1名分が無料になる。

| 条件 | 内容 |
|---|---|
| 対象 | 国内の厳選レストラン |
| 適用条件 | 2名以上で所定のコースを予約 |
| 割引 | 1名分無料 |
| 利用回数 | 各店舗につき利用期間中1回 |

コース料金が1名15,000円の場合、1回の利用で15,000円分の価値がある。年2〜3回利用すれば、それだけで年会費の大部分を回収できる計算。

### コンシェルジュデスク

24時間365日対応の専用コンシェルジュサービス。レストラン予約、旅行手配、ゴルフ場手配などを依頼できる。MyJCBアプリからも利用可能。

### プライオリティ・パス

世界145以上の国・地域にある1,700ヵ所以上の空港ラウンジを無料で利用できる。プライオリティ・パスのプレステージ会員相当（通常年会費$469）が付帯する。

### その他の特典

| 特典 | 内容 |
|---|---|
| 海外旅行傷害保険 | 最高1億円（自動付帯） |
| 国内旅行傷害保険 | 最高5,000万円（自動付帯） |
| ショッピング保険 | 年間500万円 |
| 国内空港ラウンジ | 無料 |
| ゴルファー保険 | 付帯 |

## インビテーションの条件

JCBはインビテーションの具体的な基準を公開していない。公式には「JCBゴールド、JCBゴールド ザ・プレミア、JCBプラチナを利用している方のうち、一定の条件を満たした方」としか明記されていない。

ただし、各種情報を総合すると以下が目安として挙げられている。

| 項目 | 目安 |
|---|---|
| 対象カード | JCBプラチナ / JCBゴールド ザ・プレミア / JCBゴールド |
| 年間利用額 | 200万円以上 |
| 利用年数 | 2年以上 |
| 累積利用額 | 1,000万円以上という説もあり |
| 支払い遅延 | なし（延滞は致命的） |
| 年齢 | 30歳以上が目安 |

**注意：上記はあくまで各種情報源からの推定であり、JCB公式が公表した基準ではない。**

### インビテーションへの最短ルート

JCBプラチナからのルートが最も早いとされている。

```
JCBプラチナ（年会費27,500円・直接申込可）
  ↓ 年間200万円以上の利用を2年程度
JCB THE CLASS（インビテーション）
```

JCBゴールドからの場合は、ゴールド → ゴールド ザ・プレミア → ザ・クラスとステップが増えるため、期間が長くなる。

## JCBプラチナとの比較

| 項目 | JCB THE CLASS | JCBプラチナ |
|---|---|---|
| 年会費 | 55,000円 | 27,500円 |
| 申込方法 | インビテーションのみ | 直接申込可 |
| メンバーズ・セレクション | ○ | × |
| グルメ・ベネフィット | ○ | ○ |
| コンシェルジュ | ○（24/365） | ○（24/365） |
| プライオリティ・パス | ○ | ○ |
| 旅行傷害保険（海外） | 最高1億円 | 最高1億円 |

グルメ・ベネフィット、コンシェルジュ、プライオリティ・パスはプラチナでも利用可能。ザ・クラス固有の特典はメンバーズ・セレクションとカードのステータス。年会費の差額27,500円に対して、メンバーズ・セレクション（約25,000円相当）でほぼ相殺される計算。

## まとめ

JCB THE CLASSは年会費55,000円のJCB最高峰カード。メンバーズ・セレクション（約25,000円相当）とグルメ・ベネフィット（1回で15,000円〜の価値）を活用すれば、年会費以上のリターンが見込める。インビテーションの目安はJCBプラチナで年間200万円以上の利用を2年程度。招待制であること自体がこのカードの価値でもある。
