## ラグジュアリーカード ブラックとは

ラグジュアリーカード ブラックは、新生銀行グループのアプラスが発行する金属製クレジットカード。マットブラックのステンレスカードは約22gで、[チタン](/creditcard/luxury-card-titanium/)のシルバー仕上げとは異なる重厚感がある。Mastercard最上位グレード「World Elite」を搭載し、3ランク（チタン・ブラック・ゴールド）の中間に位置する。

年会費110,000円はチタン（55,000円）の2倍だが、還元率1.25%への引き上げ、映画毎月2回、リムジン送迎の追加など、特典の上乗せ幅を見て元が取れるかが判断の分かれ目になる。

## チタンとの差分

年会費が55,000円上がることで何が変わるかを整理する。

| 項目 | チタン（55,000円） | ブラック（110,000円） |
|---|---|---|
| カード色 | シルバー（ブラッシュド） | マットブラック |
| ポイント還元率 | 1.0% | 1.25% |
| キャッシュバック | 1.0% | 1.25% |
| 景品交換最大還元 | 3.6% | 4.5% |
| マイル還元率 | 0.6% | 0.75% |
| 映画特典 | 毎月1回 | 毎月2回 |
| リムジン送迎 | なし | あり（片道） |
| ハワイアン航空ステータス | なし | Pualani Gold |
| コンシェルジュ | あり | あり |
| ダイニング1名無料 | あり（250店以上） | あり（250店以上） |
| プライオリティパス | プレステージ | プレステージ |
| ホテル特典（70,000円相当） | あり | あり |
| カフェ毎月1杯無料 | あり | あり |
| 家族カード年会費 | 16,500円 | 27,500円 |

コンシェルジュ・ダイニング1名無料・プライオリティパス・ホテル特典はチタンと同等。差額55,000円の価値は、還元率の上乗せ・映画2回・リムジン送迎に集約される。

## ポイント還元率

| 用途 | 還元率 |
|---|---|
| 通常利用 | 1.25% |
| 景品交換 | 最大4.5% |
| キャッシュバック | 1.25% |
| マイル交換（ANA/JAL/ユナイテッド） | 0.75% |

チタンの1.0%から0.25%の上乗せ。年間200万円利用で差額は5,000円、400万円で10,000円。還元率の差だけで年会費差額55,000円を埋めるには年間2,200万円の利用が必要になるため、還元率目的だけでアップグレードするのは合理的ではない。

## 主要特典

### コンシェルジュサービス

24時間365日対応の専任コンシェルジュ。電話・メール・LINEで利用可能。チタンと同じサービスだが、ブラック以上ではコンシェルジュの対応品質が上がるという口コミもある（公式には明言されていない）。

### ダイニング特典（1名無料）

全国250以上の対象レストランで、所定のコース料理を2名以上で予約すると1名分が無料。さらに6名予約で3名分が無料になる特典もある。1回あたり10,000〜16,000円相当の価値があり、年に5回使えば50,000〜80,000円分を回収できる。カフェで毎月1回、対象ドリンクが1杯無料になる特典も付帯。

### リムジン送迎（ブラック以上）

対象レストランへの片道リムジン送迎が無料で利用できる。チタンにはない特典で、1回あたり約10,000円相当の価値。特別なディナーの体験価値を引き上げる所有体験型の特典として、ラグジュアリーカードの象徴的なサービスのひとつ。

### 映画特典（毎月2回）

毎月2枚の映画鑑賞GIFTが提供される。TOHOシネマズなど主要シネコンで利用可能。年間で約48,000円相当（2,000円×24枚）の価値があり、チタンの毎月1回（年間24,000円相当）から倍増する。

### プライオリティパス

148カ国・600都市以上、1,700カ所以上の空港ラウンジを無料で利用できるプレステージ会員。チタンと同等。

### ホテル特典

世界5,000軒以上の対象ホテルで、1滞在あたり約70,000円相当の特典。ルームアップグレード・朝食無料・レイトチェックアウトなどが含まれる。チタンと同等。

### ハワイアン航空上級会員（ブラック以上）

ハワイアン航空のPualani Gold会員資格が無条件で付与される。優先チェックイン・優先搭乗・預け荷物の優先受け取りなどの特典がある。ハワイ旅行の頻度が高い場合に価値が出る。

保険の補償額はチタンと同等。海外旅行傷害保険はキャッシュレス診療対応で、家族特約（6親等以内の血族・3親等以内の姻族）も付帯する。

## 年会費110,000円の損益分岐点

| 特典 | 年間の還元/節約額 |
|---|---|
| ダイニング1名無料（年5回） | 約50,000〜80,000円 |
| 映画毎月2回無料（年24回） | 約48,000円 |
| プライオリティパス（年3回） | 約15,000円相当 |
| リムジン送迎（年3回） | 約30,000円相当 |
| カフェ毎月1杯無料（年12回） | 約6,000円 |
| 合計 | 約149,000〜179,000円 |

ダイニング・映画・リムジンを積極的に使えば年会費の1.5倍程度のリターンが見込める。特に映画特典が毎月2回に倍増する点は、映画好きにとっては年間24,000円分の追加価値になる。

### チタンとの損益比較

チタン（55,000円）の損益分岐点は約85,000〜109,000円。ブラック（110,000円）は約149,000〜179,000円。差額55,000円に対して追加リターンは約64,000〜70,000円（映画+24,000円、リムジン+30,000円、還元率上乗せ+α）なので、映画とリムジンを実際に使うなら差額以上の回収が可能。

## JCBプラチナとの比較

年会費が近い[JCBプラチナ](/creditcard/jcb-platinum/)（27,500円）と比較する。年会費は4倍の差があるが、特典構成は大きく異なる。

| 項目 | ラグジュアリーカード ブラック | JCBプラチナ |
|---|---|---|
| 年会費 | 110,000円 | 27,500円 |
| 国際ブランド | Mastercard | JCB |
| ポイント還元率 | 1.25% | 0.5%〜 |
| コンシェルジュ | あり（24時間） | あり（24時間） |
| ダイニング1名無料 | あり（250店以上） | あり（グルメ・ベネフィット） |
| 空港ラウンジ | プライオリティパス | プライオリティパス |
| 映画特典 | 毎月2回無料 | なし |
| リムジン送迎 | あり | なし |
| カード素材 | 金属（ステンレス） | プラスチック |
| 上位カード | ゴールド（220,000円） | ザ・クラス（インビテーション） |

JCBプラチナはコンシェルジュとダイニング1名無料を27,500円で実現するコスパ最強のカード。ラグジュアリーカード ブラックは映画・リムジン・金属カード・高還元率で差別化される。年会費差82,500円に見合う価値を感じるかは利用スタイル次第。

## 注意点

| 項目 | 内容 |
|---|---|
| 国際ブランド | Mastercardのみ（Visa不可） |
| 家族カード | 27,500円/枚（チタンの16,500円より高い） |
| マイル還元率 | 0.75%と低め（マイラーには不向き） |
| カード重量 | 約22gで重い（自動改札・セルフレジで引っかかる場合あり） |
| ポイント有効期限 | 最大5年（無期限ではない） |

| リムジン | 往路のみ（復路は対象外） |

## まとめ

ラグジュアリーカード ブラックは、年会費110,000円でチタンの全特典に加え、還元率1.25%・映画毎月2回・リムジン送迎・ハワイアン航空ステータスが上乗せされる。差額55,000円に対して、映画とリムジンだけで54,000円以上の追加リターンが見込めるため、これらの特典を実際に使うなら合理的なアップグレード。逆に映画やリムジンに興味がなければ、[チタン](/creditcard/luxury-card-titanium/)で十分。マットブラックの金属カードという所有体験に価値を感じるかどうかも、判断基準のひとつになる。
