## ラグジュアリーカード ゴールドとは

ラグジュアリーカード ゴールドは、新生銀行グループのアプラスが発行する金属製クレジットカードの最上位モデル。24金コーティングのカード本体は約22gで、[チタン](/creditcard/luxury-card-titanium/)のシルバー、[ブラック](/creditcard/luxury-card-black/)のマットブラックとは異なる圧倒的な存在感がある。日本限定でローズゴールドも選択可能。Mastercard最上位グレード「World Elite」を搭載し、3ランクの頂点に位置する。

年会費220,000円はブラック（110,000円）の2倍だが、還元率1.5%への引き上げ、映画毎月3回、リムジン往復対応、ハワイアン航空Pualani Platinumへの昇格など、特典の上乗せ幅を見て元が取れるかが判断の分かれ目になる。

## ブラックとの差分

年会費が110,000円上がることで何が変わるかを整理する。

| 項目 | ブラック（110,000円） | ゴールド（220,000円） |
|---|---|---|
| カード色 | マットブラック | 24金ゴールド / ローズゴールド |
| ポイント還元率 | 1.25% | 1.5% |
| キャッシュバック | 1.25% | 1.5% |
| 景品交換最大還元 | 4.5% | 5.4% |
| マイル還元率 | 0.75% | 0.9% |
| 映画特典 | 毎月2回 | 毎月3回 |
| リムジン送迎 | 片道（往路のみ） | 往復対応 |
| ハワイアン航空ステータス | Pualani Gold | Pualani Platinum |
| HoteLux | エリート | エリートプラス |
| コンシェルジュ | あり | あり |

コンシェルジュ・ダイニング1名無料・プライオリティパス・ホテル特典はブラックと同等。差額110,000円の価値は、還元率の上乗せ・映画3回・リムジン往復・ハワイアン航空Pualani Platinum・HoteLuxエリートプラスに集約される。

## ポイント還元率

| 用途 | 還元率 |
|---|---|
| 通常利用 | 1.5% |
| 景品交換 | 最大5.4% |
| キャッシュバック | 1.5% |
| マイル交換（ANA/JAL/ユナイテッド） | 0.9% |

ブラックの1.25%から0.25%の上乗せ。年間300万円利用で差額は7,500円、500万円で12,500円。還元率の差だけで年会費差額110,000円を埋めるには年間4,400万円の利用が必要になるため、還元率目的だけでアップグレードするのは合理的ではない。

マイル還元率は0.9%で、3ランク中最も高い。ANA・JAL・ユナイテッド航空への交換手数料は無料で、交換上限もない。マイラー視点でもラグジュアリーカードの中では最も効率が良い。

## 主要特典

### コンシェルジュサービス

24時間365日対応の専任コンシェルジュ。電話・メール・LINEで利用可能。ゴールドはコンシェルジュの対応品質が最も高いとされ、より複雑なリクエストへの対応力が期待できる（公式には明言されていない）。

### ダイニング特典（1名無料）

全国250以上の対象レストランで、所定のコース料理を2名以上で予約すると1名分が無料。6名予約で3名分が無料になる特典もある。1回あたり10,000〜16,000円相当の価値があり、年に6回使えば60,000〜96,000円分を回収できる。カフェで毎月1回、対象ドリンクが1杯無料になる特典も付帯。

### リムジン送迎（往復対応）

対象レストランへのリムジン送迎が無料で利用できる。ブラックは片道（往路のみ）だが、ゴールドは往路・復路どちらにも対応する。1回あたり約10,000円相当の価値で、往復利用なら約20,000円相当。最大4名まで同乗可能で、空港・オフィス・駅など対象エリア内であればどこからでも送迎が可能。海外ではニューヨーク・ロサンゼルス・サンフランシスコ・シンガポール・上海のレストランが対象。

### 映画特典（毎月3回）

毎月3枚の映画鑑賞GIFTが提供される。TOHOシネマズなど主要シネコンで利用可能。年間で約72,000円相当（2,000円×36枚）の価値があり、ブラックの毎月2回（年間48,000円相当）からさらに24,000円分が上乗せされる。

### プライオリティパス

148カ国・600都市以上、1,700カ所以上の空港ラウンジを無料で利用できるプレステージ会員。ブラック・チタンと同等。

### ホテル特典

世界5,000軒以上の対象ホテルで、1滞在あたり約70,000円相当の特典。ルームアップグレード・朝食無料・レイトチェックアウトなどが含まれる。ブラック・チタンと同等。

### HoteLuxエリートプラス（ゴールド限定）

通常年会費約75,000円相当の「HoteLux」エリートプラス会員資格が無料付帯。マリオット・ヒルトン系を含む高級ホテルで、部屋のアップグレード・朝食・レイトチェックアウトなどの優待が受けられる。ブラック以下の「エリート」と比べ、アップグレードの優先度が上がる。

### ハワイアン航空上級会員（Pualani Platinum）

ハワイアン航空の最上位ステータス「Pualani Platinum」が無条件で付与される。ブラックのPualani Goldと比較して、ラウンジアクセス・優先搭乗・座席アップグレードの優先度がさらに上がる。ハワイ旅行の頻度が高い場合に大きな価値を持つ。

## 年会費220,000円の損益分岐点

| 特典 | 年間の還元/節約額 |
|---|---|
| ダイニング1名無料（年6回） | 約60,000〜96,000円 |
| 映画毎月3回無料（年36回） | 約72,000円 |
| プライオリティパス（年3回） | 約15,000円相当 |
| リムジン送迎（年3回・往復） | 約60,000円相当 |
| HoteLux（年会費相当） | 約75,000円相当 |
| カフェ毎月1杯無料（年12回） | 約6,000円 |
| 合計 | 約288,000〜324,000円 |

ダイニング・映画・リムジン・HoteLuxをフル活用すれば年会費の1.3〜1.5倍のリターンが見込める。特にリムジンが往復対応になることで、ブラックの片道（年3回で約30,000円）から約60,000円に倍増する点が大きい。

### ブラックとの損益比較

ブラック（110,000円）の損益分岐点は約149,000〜179,000円。ゴールド（220,000円）は約288,000〜324,000円。差額110,000円に対して追加リターンは映画+24,000円、リムジン復路追加+30,000円、HoteLuxエリートプラス+75,000円相当で約129,000円。特典をフル活用する前提なら差額以上の回収は可能だが、使わない特典が増えるほど費用対効果は急激に悪化する。

## まとめ

ラグジュアリーカード ゴールドは、年会費220,000円でブラックの全特典に加え、還元率1.5%・映画毎月3回・リムジン往復・ハワイアン航空Pualani Platinum・HoteLuxエリートプラスが上乗せされる。差額110,000円に対して、特典をフル活用すれば約129,000円の追加リターンが見込めるため、数字上は回収可能。ただし、リムジン往復やHoteLuxを実際に年間複数回使うかどうかが分かれ目で、使わない特典が多ければ[ブラック](/creditcard/luxury-card-black/)のほうが費用対効果は高い。24金コーティングの所有体験に価値を感じるかどうかも、ゴールドならではの判断基準になる。
