## Pixel 10aの基本情報

Googleが2026年4月14日に「Pixel 10a」を日本で発売する。予約開始は4月8日。Pixelの「a」シリーズはフラッグシップの技術を手の届く価格に落とし込んだモデルで、Pixel 10aも同様の位置づけだ。

プロセッサはTensor G4を搭載。フラッグシップのPixel 10がTensor G5を採用しているのに対し、10aは前世代チップとなる。ただしAI処理やカメラの計算写真機能はTensor G4でも十分に動作する。

## Pixel 9aからの進化点

| 項目 | Pixel 10a | Pixel 9a |
|---|---|---|
| プロセッサ | Tensor G4 | Tensor G4 |
| ディスプレイ輝度 | HDR 2,000nits / ピーク 3,000nits | HDR 1,800nits / ピーク 2,700nits |
| ディスプレイガラス | Gorilla Glass 7i | Gorilla Glass 3 |
| 有線充電 | 30W | 23W |
| ワイヤレス充電 | 10W | 7.5W |
| 衛星SOS | 対応 | 非対応 |
| バッテリー | 5,100mAh | 5,100mAh |
| 価格 | 約79,900円〜 | 79,900円 |

プロセッサは同じTensor G4で据え置き。進化の中心はディスプレイ（輝度向上＋Gorilla Glass 7i）、充電速度（有線30W＋無線10W）、衛星SOS対応の3点だ。劇的なスペックアップというよりは、弱点を順当に改善したアップデートになっている。

## 予約特典：Google Storeポイント

Google Storeで購入すると5,000円分のストアポイントが付与されるキャンペーンが実施されている。ストアポイントはGoogle Storeでの次回購入に使えるため、ケースやイヤホンなどのアクセサリー購入に充てられる。

キャリア（docomo・au・SoftBank）での取り扱いも想定されるが、キャリア版の場合は独自のキャンペーンが適用される可能性がある。Google Store直販で購入するか、キャリアの割引を使うかは、各キャリアのキャンペーン内容が出揃ってから比較するのがよい。

## Pixel 10aは買いか

Pixel 10aの立ち位置は明快だ。約8万円でTensor G4のAI機能、120Hz OLED、48MPカメラ、5,100mAhバッテリーが手に入る。同価格帯のAndroidスマートフォンと比較しても、カメラの計算写真処理はPixelが頭一つ抜けている。

Pixel 9aからの買い替えは正直微妙だ。プロセッサが同じTensor G4で、体感速度の差はほぼない。ディスプレイの耐久性と充電速度は改善されているが、それだけのために8万円を出す理由は薄い。

一方、Pixel 7a以前から買い替える場合は明確にスペックアップを実感できる。Gorilla Glass 7iによる耐久性向上、30W充電による充電時間短縮、衛星SOS対応は実用的な進化だ。

## まとめ

Pixel 10aは「aシリーズとしての完成度が上がった順当進化モデル」という評価が妥当だ。4月8日の予約開始に向けて、Google Storeの5,000ポイント特典を活用するか、キャリアのキャンペーンを待つかが判断ポイントになる。Pixel 9aユーザーは急ぐ必要なし、Pixel 7a以前のユーザーには良い買い替えタイミングだ。
